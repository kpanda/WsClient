Lets make WS simple

About

This simple tool helps us to test any web services. It uses jQuery for functional or operational and jQuery UI for its rich and flexible UI. We encourage people to fork / download / create new repo out of this repo.

Official site

The official site of this tool is: WsClient site.

Download

You can visit the WsClient repo in GIT Hub. The source code can be downloaded in zipped file.
